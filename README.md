# MBTIRPG

This repository implements basic personality scoring utilities for games or applications.

## Features

- **MBTI Scoring** – track points for the four dichotomies (E/I, S/N, T/F, J/P) and obtain a 4-letter type code with English and Chinese descriptions.
- **Enneagram Scoring** – accumulate scores for all nine Enneagram types.
- **Big Five (OCEAN) Scoring** – optional tracking for openness, conscientiousness, extraversion, agreeableness and neuroticism.

See `personality.py` for the implementation and run tests with:

```bash
pytest
```
