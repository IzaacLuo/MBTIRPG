# MBTIRPG

A Flutter-based MBTI-themed role playing game.

## Build and Run

1. [Install Flutter](https://docs.flutter.dev/get-started/install) and set up your development environment.
2. Fetch dependencies:
   ```sh
   flutter pub get
   ```
3. Generate localization files:
   ```sh
   flutter gen-l10n
   ```
4. Run the app on a connected device or emulator:
   ```sh
   flutter run
   ```
5. To build release binaries:
   - Android APK: `flutter build apk`
   - iOS app: `flutter build ios`

The project includes localization scaffolding for English and Chinese. Edit files in `lib/l10n/` to add or update translations.
