# MBTIRPG

This repository contains resources for the **MBTIRPG** project.

## Localization

Translation files live in `assets/i18n`. Each locale has its own JSON file. To contribute a translation:

1. Copy `assets/i18n/en.json` to a new file named with your locale, such as `fr.json` for French.
2. Replace the English text with your translated strings.
3. Submit a pull request with the new or updated JSON file.

## Contributing Stories or Tests

New stories or tests are welcome! Add your files to the appropriate directories (for example `stories/` or `tests/`) and open a pull request explaining the additions.
